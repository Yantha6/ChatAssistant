嫦娥的信息

技能信息:
被动 - 月盈:
描述: 嫦娥每1点法术攻击会提升1.5点最大法力，受到伤害的60%优先由法力承担，每1点法力值抵挡2点伤害。嫦娥的法力越高造成的伤害越高，满法力时可增加100%伤害。
冷却值: 0
消耗: 0

技能1 - 月辰:
描述: 嫦娥向指定方向释放能量，对命中的第一个目标造成150/180/210/240/270/300（+25%法术加成）点法术伤害并标记敌方3秒同时造成1秒定身效果，定身时长随法术强度影响，每100点法术强度额外造成0.15秒定身，最高定身2.5秒；命中后增加嫦娥30%移速持续3秒。嫦娥的普攻与技能伤害能够引爆标记，对范围内的敌人造成150/180/210/240/270/300（+25%法术加成）点法术伤害和50%减速持续1秒；被动：嫦娥可以非常轻松的引动游离在身边的能量 ，让普通攻击变为法术伤害造成80点法术伤害，每过5秒的第一次普攻射程提升并会造成额外80（+10%法术加成）点法术伤害并回复自身最大蓝量5%
冷却值: 12/11.2/10.4/9.6/8.8/8.0
消耗: 0

技能2 - 月璇:
描述: 嫦娥在身边和指定位置召唤法力旋涡持续4秒，每次消耗范围内敌人15/20/25/30/35/40点法力值并回复自身0.5～1%最大法力，对小兵额外回蓝20%，对野怪减少50%，造成30/37/44/51/58/65（+7%法术攻击）法术伤害。敌方法力不足时，不足消耗的法力将转成对其造成同等的法术伤害（对小兵额外造成50%伤害，对野怪不生效）。
冷却值: 8
消耗: 0

技能3 - 月芒:
描述: 嫦娥解放狂暴的魔道力量提升自身30%移速，疯狂倾泻自身的法力值并随机向周围发射法力弹幕，每颗弹幕消耗48点法力，升级提升每秒弹幕数量（8/12/16）对命中的敌人造成100点法术伤害
冷却值: 8
消耗: 0

装备推荐:

Tips：嫦娥大量倾泻法力值是可以轻松造成高额的范围伤害，如果队友能够支撑其前排给予输出环境，嫦娥可以选择全法攻出装

英雄关系:
最佳搭档:
嫦娥大招爆发范围不小，但作为法师来说还是太近了，需要鬼谷子的帮助，让其靠近敌人 嫦娥虽然坦度不错，但是并不能作为主动吸收伤害的角色，在团队里最好在配置一位坦克英雄进行开团和吸收伤害
压制英雄:
上官婉儿的大招移动路径比较容易预判，嫦娥的1技能即使再缓慢还是能够逼退 李白可以给予后排法师极大的压力，嫦娥虽然也是法师，但具有坦克的属性，不会被李白的刺杀轻易的压制
被压制英雄:
伽罗拥有超远的射程和强势减速能力，这让嫦娥入场十分困难，并且在进入战斗前就会把嫦娥的蓝量削减至可以承受的范围内 吕布真实伤害可以越过嫦娥的护盾直接削减嫦娥的血量，非常克制依赖护盾减伤的嫦娥
