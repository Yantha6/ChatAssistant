吕布的信息

技能信息:
被动 - 饕餮血统:
描述: 吕布挥斩方天画戟的技能命中敌方英雄后会给武器附魔，持续8秒。方天画戟在附魔状态下造成的是等额的真实伤害。附魔状态下，吕布挥斩方天画戟的技能每命中一名敌方英雄，回复(+50％额外物理攻击)(+4%已损生命)点生命。附魔状态下，吕布每次普攻命中敌人，回复(+25％额外物理攻击)(+1％已损生命)点生命。
冷却值: 0
消耗: 0

技能1 - 方天画斩:
描述: 吕布向前挥斩方天画戟，造成500/635/770/905/1040/1175(+260％额外物理攻击)物理伤害。如果挥斩没有命中敌方英雄，则方天画戟会立即失去附魔。
冷却值: 5
消耗: 0

技能2 - 贪狼之握:
描述: 吕布夺取范围内敌人的魂灵，造成30%减速，持续3秒。每夺取一个魂灵，吕布获得175/220/265/310/355/400(+150％额外物理攻击)点护盾，护盾最多叠加3层。吕布每秒将当前25％的护盾转化为生命，每2点护盾转化为1点生命。
冷却值: 8
消耗: 0

技能3 - 魔神降世:
描述: 吕布从天而降，圈定杀戮场展开杀戮，对范围内的敌人造成1秒击飞。落地后吕布奋力挥斩方天画戟，造成550/800/1050(+300％额外物理攻击)物理伤害。吕布在杀戮场内获得150/200/250点物理和法术防御。触碰场地边缘的敌人会受到150/200/250(+87％额外物理攻击)物理伤害和50％减速，持续2秒，若为英雄则还会为吕布额外回复(+50％额外物理攻击)(+4％已损生命)点生命。
冷却值: 50
消耗: 0

技能加点建议:
主生: 火焰三尖枪
副生: 混天绫.束
召唤师技能: 闪现/斩杀

铭文信息:
圣人 法术攻击力+5.3
虚空 最大生命+37.5 冷却缩减+0.6%
轮回 法术攻击力+2.4 法术吸血+1%
铭文建议:
Tips：吕布作为一名近战英雄，物理攻击及生存能力至关重要，移速的增加能提高吕布的支援速度，在战场上以更灵活的身姿作战，6%的冷却缩减可让吕布更频繁地使用技能去打出更多的伤害

装备2: 冷静之靴
装备3: 圣杯
装备4: 极寒风暴
装备5: 梦魇之牙

装备推荐:
Tips：吕布的技能伤害加成是额外物理攻击加成，破军是该属性最高的装备，同时吕布的伤害主要来源是附魔后的普攻真实伤害，所以在中期需要补充一定的攻击速度以保证可以打出足够多的普攻次数
Tips：吕布的大招拥有极大的威慑力，极寒风暴能让吕布每次团战都会有准备好的大招

英雄关系:
最佳搭档:
武则天：武则天的连续控制及伤害可以较好的配合吕布进行输出。团战中吕布先手开大控制敌方后排，武则天在再开大造成控制和成吨输出。 白起：白起在团战中是一个主坦的位置，主要在团战中承受伤害、控制敌方英雄，而吕布这样一个偏进攻型的战士则是非常需要一个控制留人的英雄打配合。
压制英雄:
张飞：张飞大招变身和二技能都能为自己套上一个较厚的护盾，但附魔时的吕布为真实伤害无视护盾。 黄忠：黄忠是个移动炮台，但开大后的黄忠无法移动，此时吕布使用大招，造成控制和伤害，配合队友轻松击杀黄忠。
被压制英雄:
项羽的1技能可以有效打断吕布施放大招 王昭君的减速和控制技能很好风筝吕布
